# ⚕️ Pn4 - 지능형 개인 라이프스타일 오거나이저 (Serverless)

본 프로젝트는 레오(LEOHYP)님의 개인 일상을 풍요롭게 하기 위한 **여행 계획 수립, 맛집 및 여정 기록, 시각화 아카이빙 대시보드**입니다. 
백엔드 서버 구동 없이 브라우저 단독으로 동작하는 **100% 서버리스(Serverless)** 독립형 웹 애플리케이션으로 설계되었습니다.

---

## 🌟 주요 특징 (Features)

*   **100% 클라이언트 사이드 구동**: 별도의 파이썬 백엔드(Flask 등)나 Node.js 설치가 전혀 필요 없습니다. 브라우저에서 `index.html` 파일을 열기만 하면 즉시 실행됩니다.
*   **Obsidian 로컬 보관소 직접 연동**: 브라우저 표준 **File System Access API**를 통해, 로컬 PC에 존재하는 레오님의 Obsidian 다이어리 폴더(`LifeStyle-Diary`)를 직접 읽고 씁니다.
*   **안전한 API Key 관리**: Gemini API Key는 외부로 송출되거나 유출되지 않고, 오직 브라우저의 로컬 보안 영역인 `localStorage`에만 저장 및 호출됩니다.
*   **마크다운 렌더링**: 수동으로 폼에 입력해 저장한 후기나 Gemini가 실시간으로 수립해 준 여행 계획서가 모두 아름다운 YAML Frontmatter를 동반한 마크다운 파일(`.md`)로 로컬에 바로 저장되며, 대시보드에서도 Marked.js를 통해 마크다운 뷰어로 즉시 열어볼 수 있습니다.

## ⚙️ 실행 및 사용 방법 (How to Run)

### 1단계: 브라우저에서 열기
*   Chrome(크롬) 또는 Edge(엣지) 브라우저를 열고, `index.html` 파일을 **마우스로 드래그 앤 드롭**하거나 **더블클릭**하여 실행합니다.

### 2단계: Obsidian 로컬 폴더 연동
1.  왼쪽 사이드바의 **[보관소 폴더 연결]** 버튼을 클릭합니다.
2.  Obsidian에서 개인 일기나 계획표가 보관될 폴더(예: `C:\Users\ismadmin\Documents\ObsidianVault\LifeStyle-Diary`)를 선택합니다.
3.  브라우저 상단에 "파일 보기 권한 허용" 및 "파일 변경 허용" 팝업이 뜨면 **[권한 허용(또는 변경 사항 저장)]**을 승인해 줍니다.
4.  *이후 새로고침하더라도 IndexedDB를 통해 폴더 정보가 복원되며, 보안 정책상 최초 1회만 클릭하여 잠금 해제(권한 재승인)해 주시면 됩니다.*

### 3단계: Gemini API 설정
1.  사이드바 좌측 하단의 **[Gemini 설정]** 기어 아이콘을 누릅니다.
2.  레오님의 **Gemini API Key**를 입력한 뒤 **[저장하기]**를 누르면 모든 준비가 끝납니다.
