# Agent Team

여러 [[AI 에이전트]]가 특정 목표를 달성하기 위해 서로 협력하고 소통하는 구조입니다. [[하네스 엔지니어링]]에서 복잡하고 논의·협업이 필요한 작업에 주로 사용됩니다.

[[SubAgent]]와 비교했을 때 다음과 같은 특징이 있습니다:

*   **소통 방식**: 팀원끼리 직접 메시지를 주고받으며 소통합니다.
*   **조율 방식**: 공유 작업 목록을 통해 작업을 조율합니다.
*   **적합한 작업**: 논의와 협업이 필요한 복잡한 작업에 효율적입니다.
*   **비용**: [[SubAgent]]보다 토큰 비용이 더 큽니다.

[[Claude Code]] 환경에서 [[Agent Team]] 기능을 활성화하려면 `~/.claude/settings.json` 파일에 `"CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS": "1"` 환경 변수를 추가해야 합니다. (업데이트: 2026-06-19 16:27:21)
