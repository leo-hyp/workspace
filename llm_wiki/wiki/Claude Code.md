# Claude Code

[[Anthropic]]에서 개발한 대규모 언어 모델(LLM) 기반의 코드 에이전트 환경으로, [[하네스 엔지니어링]]과 [[Dynamic Workflow]]의 구현에 활용되는 플랫폼입니다. [[Andrej Karpathy]]의 [[LLM Wiki]] 개념에서 LLM "프로그래머"의 "IDE" 역할을 하는 [[Obsidian]]과 유사하게, [[AI 에이전트]]가 코드를 생성하고 관리하는 주요 도구로 사용될 수 있습니다.

특히 [[Dynamic Workflow]] 기능을 통해 [[AI 에이전트]]의 협업과 복잡한 작업을 체계적으로 설계하고 실행할 수 있도록 지원하며, 개발자의 체감 속도를 크게 향상시키는 데 기여합니다. (업데이트: 2026-06-19 16:27:21)
