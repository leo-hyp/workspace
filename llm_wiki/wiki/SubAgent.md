# SubAgent

[[AI 에이전트]]의 한 형태로, 메인 에이전트의 관리 하에 특정 단일 작업을 빠르게 수행하는 데 적합합니다.

[[Agent Team]]과 비교했을 때 다음과 같은 특징이 있습니다:

*   **관리 주체**: 메인 에이전트가 직접 관리합니다.
*   **결과 회수**: 결과를 빠르게 회수하여 전달합니다.
*   **적합한 작업**: 집중된 단일 작업에 효율적입니다.
*   **비용**: [[Agent Team]]보다 토큰 비용이 적게 듭니다.

빠른 워커가 필요한 소규모 단일 작업에 주로 활용됩니다. (업데이트: 2026-06-19 16:27:21)
