# 🏥 병원 보안 & IT 뉴스 브리핑 센터 (Ver 1.5)

병원 보안 및 IT 의사결정권자를 위한 고성능 실시간 인텔리전스 대시보드입니다.

## 👥 개발자 정보
- **Lead Developer & Product Owner:** LEOHYP
- **AI Development Partner:** Antigravity AI (Google DeepMind)

## 🚀 주요 기능
1. **무중단 실시간 뉴스 스트리밍**: 30분 주기로 자동 갱신되며, 백그라운드 동기화를 통해 화면 끊김 없는 정보를 제공합니다.
2. **지능형 뉴스 분류**: 수집된 뉴스를 [보안/사고], [IT/기술], [병원/의료], [사회] 카테고리로 자동 분류합니다.
3. **실시간 금융 지표 연동**: 삼성전자, SK하이닉스, USD/KRW 환율 정보를 실시간으로 추적하여 대시보드 우측에 표시합니다.
4. **개인화 관심기사(❤️)**: 중요한 기사는 하트 버튼으로 스크랩하여 별도 탭에 영구 보관할 수 있습니다.
5. **프리미엄 UI/UX**: 슬레이트 네이비 테마와 다크모드를 지원하며, 상단 브리핑 존을 통해 주요 소식을 실시간으로 롤링합니다.

## 🛠 기술 스택
- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Data Engine:** RSS2JSON API, Yahoo Finance RSS
- **Storage:** LocalStorage (사용자 설정 및 관심기사 보관)

## 📝 설치 및 실행 방법
1. 본 레포지토리의 모든 파일을 다운로드합니다.
2. `index.html` 파일을 크롬(Chrome) 또는 엣지(Edge) 브라우저로 실행합니다.
3. 상단의 `[➕ 채널 추가]` 버튼을 통해 원하는 뉴스 소스를 자유롭게 확장할 수 있습니다.
