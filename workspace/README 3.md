# 🌌 3단원: 자율 주행 (/grill-me & /goal)

이 단원에서는 기획 단계의 모호성을 지능적으로 걷어내는 `/grill-me`와, 장기 과제를 전적으로 위임해 자율 수행하게 하는 `/goal` 슬래시 커맨드를 익힙니다.

---

## 💡 이론 1: `/grill-me` (선택지식 조율 인터뷰)

사장님이 "A 기능을 대충 이런 느낌으로 짜줘"라고 요청했을 때, 에이전트가 제멋대로 상상하여 엉뚱한 코드를 짜는 경우가 있습니다.
이때 `/grill-me`를 사용하면 안티그래비티가 사장님께 **역으로 인터뷰를 신청**합니다.
* 아키텍처 의사결정, 변수명 정책, DB 구조 등의 선택지를 A/B/C 유형으로 제공하며 하나씩 확인합니다.
* 사장님과의 조율이 끝난 완벽한 정답으로 계획을 설계하므로 개발의 정확도가 200% 증가합니다.

---

## 💡 이론 2: `/goal` (자율 실행 모드)

작업이 무겁고 수많은 단계(예: 소스 10개 파일 전체 수정, 유닛 테스트 통과 등)가 필요해 밤새 돌려야 하거나 자리를 비워야 할 때 유용합니다.
* 일반 모드에서는 도구 승인이나 추가 질문을 기다리며 실행이 대기되지만, `/goal` 모드에서는 **모든 단계의 도구(명령어, 코드 편집 등) 실행이 자동 승인**되며 최종 목표 달성 시까지 스스로 루프를 돌며 임무를 완수합니다.

---

## 🎯 실습 미션 3: 데이터 분석기 기획 조율 및 자율 개발

[mission_goal.py](file:///c:/Users/ismadmin/Documents/Workspace/Pn7_Mastery/chapters/chapter3_slash_commands/mission_goal.py) 파일은 뼈대만 있고 구현해야 할 핵심 스크립트 내용(데이터 추출, 변환, JSON 저장)이 비어 있습니다.

### 📝 미션 수행 방법
1. 대화창에 `/grill-me`를 붙여 다음과 같이 기획 인터뷰를 트리거합니다:
   > **명령 예시:** `"/grill-me Pn7_Mastery 3단원의 mission_goal.py에 가이드된 요구사항대로 데이터 분석 엔진을 완벽하게 구현하고자 해. 설계에 필요한 의사결정 인터뷰를 시작해줘."`
2. 안티그래비티가 기획(분석 알고리즘, 출력 형식 등)을 구체화하기 위해 질문을 던지는 과정을 진행해 봅니다.
3. 기획 인터뷰가 끝난 후, 안티그래비티가 마스터 계획을 세워 승인을 요청하면 **[Approve]**합니다.
4. 만약 이 작업을 자율 모드로 진행해보고 싶으시다면, 기획 종료 후 안티그래비티에게 명시적으로 `/goal`을 활용하라고 권장해 볼 수도 있습니다:
   * **명령 예시:** `"/goal 아까 확정된 계획대로 mission_goal.py의 코드를 최종 구현하고 실행 테스트까지 끝낸 뒤 성공 로그를 보여줘."`
5. 콘솔에 `[SUCCESS] Chapter 3 Mission Completed!` 로그와 함께 요약 JSON 파일이 정상 생성되면 완료됩니다.
